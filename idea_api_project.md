# 🌐 Great APIs for Teaching Async JavaScript & JSON

### (Sports, news, media, e‑commerce, gaming, and more)

Below is a **mixed‑topic list** with a **short, practical overview** of each API and **ideas for student assignments**.

***

## 🛍️ 1. Shopify Storefront API

**Type:** E‑commerce / Products  
**Difficulty:** Medium  
**Auth:** Access token required (Storefront API)

**What it does:**  
Students can fetch products, collections, images, and build simple storefronts. Supports searching, filtering, and cart creation.

**Project ideas:**

*   Build a product catalog page
*   Create a search filter for products
*   Create a simple “add to cart” experience

***

## ⛅ 2. OpenWeatherMap API

(You already use this, but still worth listing.)

**Type:** Weather  
**Difficulty:** Easy  
**Auth:** API key  
**Ideas:** Dashboards, multi‑city weather comparison, 5‑day forecast visualizations

***

## 📰 3. NewsAPI

**Type:** News / Media  
**Difficulty:** Easy  
**Auth:** API key  
**What it does:** Search headlines, topics, publishers.

**Project ideas:**

*   Build a news headlines dashboard
*   Filter articles by topic (sports/tech/health)
*   Create a “dangerous content” filter using keywords

***

## ⚽ 4. TheSportsDB

**Type:** Sports stats  
**Difficulty:** Easy  
**Auth:** Free with API key  
**What it does:** Teams, players, scores, events for many sports.

**Project ideas:**

*   Build a sports team lookup app
*   Create a “player of the day” randomizer
*   Build a scoreboard UI

***

## 🎥 5. The Movie Database (TMDB) API

**Type:** Movies / TV Shows  
**Difficulty:** Easy  
**Auth:** API key  
**What it does:** Movie details, trending lists, actors, images.

**Project ideas:**

*   Build a movie search site
*   Create a “Top 10” trending dashboard
*   Recommend movies by genre

***

## 🕹️ 6. RAWG Video Games Database API

**Type:** Video games  
**Difficulty:** Medium  
**Auth:** API key  
**What it does:** Game details, ratings, images, platforms.

**Project ideas:**

*   Build a game browser (search/filter)
*   Show upcoming or recently released games
*   “Top games by platform” page

***

## 🦠 7. Disease.sh Public Health API

**Type:** Public health / statistics  
**Difficulty:** Easy  
**Auth:** None  
**What it does:** Global data on diseases, vaccines, public health stats.

**Project ideas:**

*   Graph global or national health metrics
*   Build a “health trends” dashboard

***

## 🌌 8. NASA APIs

**Type:** Space / Science  
**Difficulty:** Easy  
**Auth:** API key (free)  
**Popular endpoints:**

*   Astronomy Picture of the Day
*   Mars Rover Photos
*   Earth imagery

**Project ideas:**

*   Display the “photo of the day”
*   Rover image gallery
*   Compare historical Earth imagery

***

## 🐱 9. GitHub REST API

**Type:** Developer / Repo data  
**Difficulty:** Medium  
**Auth:** Often optional (rate limited)  
**What it does:** Repository metadata, commits, issues.

**Project ideas:**

*   Show trending repos
*   Build a GitHub user profile viewer
*   “Top languages used” visualization

***

## 🐉 10. Pokémon API (PokeAPI)

**Type:** Games / Fun  
**Difficulty:** Easy  
**Auth:** None  
**What it does:** Pokémon stats, moves, types.

**Project ideas:**

*   Build a Pokédex
*   Filter Pokémon by type (fire/water)
*   Battle‑stat comparison tool

***

## 🍔 11. TheMealDB

**Type:** Food / Recipes  
**Difficulty:** Easy  
**Auth:** Free  
**What it does:** Recipes, ingredients, thumbnails.

**Project ideas:**

*   Random recipe generator
*   Filter by category or ingredient

***

## 🎵 12. Spotify Web API

**Type:** Music streaming metadata  
**Difficulty:** Medium/High (OAuth)  
**Auth:** OAuth tokens needed  
**What it does:** Songs, playlists, artists, albums.

**Project ideas:**

*   Build a playlist browser
*   Artist discography explorer
*   Show “top tracks” for a genre

***
